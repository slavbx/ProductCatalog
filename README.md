### ProductCatalog
Приложение для управления каталогом товаров маркетплейса       
Apache Maven 3.9.9       
Java version: 17.0.12      

### Основные возможности        
Пользовательские:
- Вход в систему и регистрация новых пользователей
- Редактирование личного профиля
- Просмотр и управление своими товарами (создание, редактирование, удаление).     

Административные:
- Просмотр списка всех зарегистрированных пользователей
- Возможность сброса паролей пользователей

### Запуск программы
- Убедитесь, что у вас установлены Java 17. Для этого выполните команду: java -version
- Откройте проект в IDE
- Нажмите кнопку Run на классе Main (в IDE IntelliJ IDEA)

### Пользование программой
- После запуска программы в консоли появится меню для входа или регистрации. Для продолжения введите команду 1, а также
свои учетные данные (email и пароль).
- После входа вы окажетесь в личном кабинете, который выполнен в виде меню. Для навигации по пунктам меню, 
а также выполнения действий и просмотра данных воспользуйтесь короткими числовыми командами, 
которые вам будут предложены.
- Для завершения работы введите команду 0 (Завершение работы) в главном меню.
